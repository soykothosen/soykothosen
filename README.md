## 🎯 Primary Research Pillars & Open-Source Tracks

### 🩺 Trustworthy AI & Medical NLP
Architecting fail-safe systems for highly critical domains, ensuring factual alignment, strict citation verification, and linguistic adaptability.
* **[TrustMed-RAG-Framework](https://github.com/Code-Studio-AI-Research-Lab/TrustMed-RAG-Framework):** Fact-checking and self-correcting RAG network built explicitly for bulletproof clinical support setups.
* **[Clinical-Banglish-Robustness](https://github.com/Code-Studio-AI-Research-Lab/Clinical-Banglish-Robustness):** Core benchmarking tools measuring language model resilience against heavy code-mixed medical token sequences.

### 🌿 AI for Science (Computational Electrophysiology & Molecular Modeling)
Expanding the frontiers of machine learning to capture and translate biological time-series data and molecular coordinates.
* **[Plant-BioSignal-Translation](https://github.com/Code-Studio-AI-Research-Lab):** Translating plants' micro-voltage electrical action potentials into natural language stress notifications.
* **[Olfactory-Language-Model](https://github.com/Code-Studio-AI-Research-Lab):** Utilizing Graph Neural Networks (GNNs) to digitize 3D molecular structures into continuous odor vector coordinates.
* **[Bio-Processing-Units-BPU](https://github.com/Code-Studio-AI-Research-Lab):** Compiling binary algorithms into nucleotide paths ($A, T, C, G$) for zero-waste, soil-biodegradable computing simulations.

### 🧠 Neuromorphic Interaction Systems & Signal Processing
Pioneering alternative input interfaces and custom speech architectures to redefine human-machine communication loops.
* **[Neuromorphic-Code-Compiler](https://github.com/Code-Studio-AI-Research-Lab):** Transformer pipelines compiling non-invasive raw EEG brainwave sequences directly into backend source code syntax.
* **[BanglaASR-Dialect-Wav2Vec](https://github.com/Code-Studio-AI-Research-Lab/BanglaASR-Dialect-Wav2Vec):** Finetuned acoustic processing networks recognizing low-resource dialects and regional language distributions.
